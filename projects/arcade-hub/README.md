# Arcade Hub

A central hub for classic games, all in one place! Arcade Hub is a static website designed to be hosted on GitHub Pages, bringing together multiple small games under one cohesive design.

## Games Included

- **Number Wordle** - Guess the secret number using logic and deduction
- **Classic Wordle** - The original word-guessing game
- **Tic-Tac-Toe** - The classic 3x3 grid game
- **Hangman** - Guess the word letter by letter
- **Minesweeper** - Navigate the minefield carefully
- **Snake** - Control the snake and grow longer
- **Memory Match** - Test your memory with matching pairs
- **Connect Four** - Drop discs and connect four in a row
- **Sudoku** - Fill the 9x9 grid with numbers
- **Blackjack** - Beat the dealer without going over 21
- **Pong** - The classic arcade game
- **Simon Says** - Follow the sequence pattern
- **Chess** - The ultimate strategy game

## Project Structure

```
arcade-hub/
├── index.html          # Main hub page with game listings
├── styles.css          # Common stylesheet for consistent design
└── README.md          # This file
```

## Design Features

- **Consistent Styling**: All games use the same `styles.css` for a cohesive look and feel
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Easy Navigation**: Clear links from the main hub to each individual game
- **Modern UI**: Clean, gradient-based design with smooth animations

## Usage

Simply open `index.html` in a web browser to view the Arcade Hub. Each game card includes a description and a "Play Now" button that links to the individual game page.

## GitHub Pages

This project is designed to be hosted on GitHub Pages. Once deployed, the main hub will be accessible at your GitHub Pages URL, and individual games can be linked from the hub.
